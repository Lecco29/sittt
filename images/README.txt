IMAGENS NECESSÁRIAS PARA O SITE ALFA CONTABILIDADE

Para que o site funcione corretamente, você precisa adicionar a seguinte imagem nesta pasta:

1. hero-bg.jpg - Imagem de fundo para todas as seções do site
   - Recomendação: Uma imagem profissional de escritório ou contabilidade
   - Tamanho recomendado: 1920x1080px ou maior
   - Formato: JPG ou PNG

Esta imagem será usada em todas as seções do site com diferentes efeitos de overlay para criar variedade visual.

Dicas para a imagem:
- Use uma imagem de alta qualidade
- Prefira uma imagem com tons claros ou neutros para não interferir na legibilidade do texto
- Certifique-se de que a imagem não tenha direitos autorais restritos
- Otimize a imagem para web para garantir carregamento rápido
